# Nos projet : Pc PortableTech
## Description
site de e-commerce spécialisé dans la vente des pc portable de haute qualité  à un prix attrctif pour les clients cele leur permet de nous contacter pour nous faire part de leurs commantaires , idées et demandes.
## Structure du projet
**backend/** : contient les fichiers PHP pour gérer la base de données et la logique serveur.
**frontend/** : contient les fichiers HTML avec php, CSS et javaScript et les photos qui nous les utilisons pour l'interface utilisateur.
**base de données/** : script SQL pour créer la base de données et les tables.
**README/** : Ce fichier.

## Installation
1. Copier le projet entier dans le dossier `htdocs` de XAMPP.
2. Ouvrir **phpMyAdmin** via XAMPP.
3. Créer une nouvelle base de données (pc_tech_boutique).
4. Importer le fichier `base-de-données.sql` dans la base de données créée.
5. Lancer **Apache** et **MySQL** depuis XAMPP.
6. Ouvrir le navigateur et accéder au projet via : http://localhost/PojetProAvWeb
